# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Git workflow

- Follow best practice: create a **feature branch** for any work that involves commits — do not commit directly to `main` (the default branch).
- When creating a branch, **announce it clearly and up front**: state the branch name and that we'll merge it back into `main` when the feature/task is done.
- When the work wraps, **flag the merge-back moment** so the user decides when to merge. Don't silently branch or merge.
- Commit or push only when the user asks.
